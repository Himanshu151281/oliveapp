❓ Troubleshooting

Images not loading?
Use a local server (Methods 2, 3, or 4) instead of direct file open for best results.
Styles look weird?
Make sure all CSS files in the css/ folder are present. Refresh the page with Ctrl+F5 (hard refresh).
Button links don't work?
Navigation to other pages (like blog, restaurants) is intentionally disabled since those pages aren't included. This is expected behavior.
